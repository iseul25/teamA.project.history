-- 연동 방법
192.168.0.120 호스트명/ip에 입력
history01 아이디
0123 비밀번호



사용자 관련 테이블
1. users (사용자 정보)
 - userId (PK=프라이머리 키)
 - userType (관리자/일반유저 구분)
 - name (이름)
 - password (비번)
 - email (UK= 유니크 키(중복방지))


출석관리 테이블
1. user_attendance (출석 기록)
 - attendanceId (PK)
 - userId (FK=외래 키/users)
 - attendanceDate (출석일자)
 - pointAdd (포인트 추가량)


게시글 관련 테이블
1. board (게시글)
 - boardId (PK)
 - userId (FK/users)
 - boardType (게시글 유형 설명란(공지사항, 학습게시글(조선시대, 삼국시대 등등)))
 - title (게시글 제목)
 - content (게시글 내용)
 - date(게시글 등록 일자)

2. board_image (게시글 이미지)
 - imgId (PK)
 - boardId (FK/board)
 - imgUrl (이미지 링크)
 - imgDescription (이미지 설명란)

2. board_comment (댓글)
 - commentId (PK)
 - boardId (FK/board)
 - userId (FK/users)
 - comment (댓글 내용)
 - date (댓글 등록 일자)


퀴즈 관련 테이블
1. quiz (퀴즈)
 - quizId (PK)
 - imgUrl (퀴즈에 쓰일 이미지 링크)
 - question (퀴즈의 질문 내용)
 - maxScore (문제별 배점)
 - createdAt (생성 시간)
 - updatedAt (수정 시간)

2. quiz_options (퀴즈 선택지)
 - optionId (PK)
 - quizId (FK/quiz)
 - optionOrder (선택지 순서: 1, 2, 3, 4) -- 화면 표시 순서 제어
 - isCorrect (정답 여부(optionOrder와 비교용))
 - optionText (선택지 내용)
 - createdAt (생성 시간)

3. quiz_score (퀴즈 점수 및 포인트 정보)
 - scoreId (PK)
 - userId (FK/users)
 - quizId (FK/quiz)
 - totalScore (총 획득 점수)
 - earnedPoints (획득 포인트)
 - createdAt (생성 시간)

4. quiz_attempt (퀴즈 응시 기록)
 - attemptId (PK)
 - sessionId (FK/quiz_session)
 - userId (FK/users)
 - quizId (FK/quiz)
 - selectedAnswer (optionOrder 값 저장) -- 사용자가 선택한 순서 번호
 - isCorrect (정답 여부(selectedAnswer와 비교용))
 - scoreEarned (획득 점수)
 - attemptTime (응시 시간)


포인트 관련 테이블
1. points (포인트 적립/차감 내역)
 - pointId (PK)
 - userId (FK/users)
 - totalPoint (총량)

2. point_transactions (포인트 사용 내역)
 - transactionId (PK)
 - userId (FK/users)
 - attendanceId (FK/users_attendance)
 - itemId (FK/point_shop)
 - scoreId (FK/quiz_score)
 - reason (변동 사유 기술)
 - pointChange (출석에 따른 포인트 증가량)
 - date (사용 일시)

3. point_shop (포인트 상점)
 - itemId (PK)
 - imgUrl (상품 이미지)
 - itemName(상품 이름)
 - cost(가격)
